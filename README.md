# cat_dog_classification

This is a cat & dog classification model using Tenserflow simply for skill development purposes.

## Datasets: 
- 200 images of dogs
- 200 images of cats

## As of 12/09/20, 

( IN PROGRESS ) 
- accuracy: 0.9187

- number of epochs: 10
- batch size: 16
